# BetterJukebox

Mod for Melody Mania
BetterJukebox is a quality-of-life mod for **Melody Mania** that transforms the Sing experience into a modern jukebox with quick controls, instant search, queue management and automatic playback.

The goal of the mod is simple:
A Simple automated Jukebox with quest and history. 

---

# Features

## Automatic playback

- Automatically opens the Sing menu when the game starts.
- Automatically selects a random song.
- One Click away from singing.

Start the party with getting people in to the grove, With mod on musik starts playing automaticly. 

---

## Better Overlay

A clean overlay that appears when moving the mouse.

Features:

- Sing
- Previous
- Play / Pause
- Next
- History
- Search
- Queue

The overlay automatically fades in and fades out when not in use.

---

## Smart Search

Search your entire song library.

Features:

- Live filtering
- Play Now
- Queue
- Scrollable results
- ESC closes search
- X button closes search
- Game shortcuts are disabled while typing

---

## Queue System

Create your own playlist while playing.

Features:

- Add songs to queue
- Remove songs
- Play queued songs
- Scrollable queue
- ESC closes queue
- X button closes queue

---

## Song History

Quickly replay songs you've recently sung.

---

## Better UI

Every popup window is designed to behave consistently.

- Responsive windows
- Never leaves the screen
- Scrollbars when needed
- ESC closes popup
- X button closes popup
- Smooth fade animations

---

## Settings

Built-in settings menu.

Current options include:

- Enable Overlay
- Auto Open Sing
- Auto Play Random Song
- Shuffle
- Show Overlay
- Fade Animations
- Auto Hide Menu
- Animation Speed
- Disable Vanilla Pause Button
- Shake Mouse To Show Menu
- Hide Mouse After Timeout

---

# Installation

Install the files to BetterJukebox folder into:

```
\AppData\LocalLow\melodymania\Melody Mania\Mods\BetterJukebox_v1_0
or 
Melody Mania\Mods\BetterJukebox_v1_0
```

Enable the mod from Melody Mania's Mod Manager.
